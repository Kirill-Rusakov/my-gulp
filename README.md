# Моя рабочая Gulp-сборка v 1.0.0

## Как получить сборку и начать работу?

### 1. Клонировать репозиторий
``` bash
git clone https://github.com/Kirill-Rusakov/my-gulp.git
```
### 2. Установить зависимости
``` bash
npm i
```
